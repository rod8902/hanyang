CORBA Compatibility Kit

RTI CORBA Compatibility Kit is an optional component that can be purchased separately. It enables applications to use both RTI Connext and CORBA without requiring conversion when going from one API to the other. RTI CORBA Compatibility Kit must be installed on an existing installation of RTI Connext 5.1.0. Please contact your account manager, RTI Sales (sales@rti.com) for purchase information.
