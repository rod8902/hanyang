Overview

RTI Distributed Logger is a library that enables applications to publish their log messages to DDS.

The log message data can be visualized with RTI Monitor, a separate GUI application that can run on the same host as your application or on a different host. Since the data is provided in a DDS topic, you can also use rtiddsspy or even write your own visualization tool.

RTI Distributed Logger also provides a remote command topic so that its behavior can be remotely controlled at run time.


API Languages

RTI Distributed Logger is provided in 3 languages:
  • C API

  • C++ API

  • Java API
